¿Qué es phpDataClass?

phpDataClass es una clase PHP para manejo de bases de datos MySQL. La idea principal de este proyecto es evitar repetirnos una y otra vez a la hora de interactuar con una base de datos MySQL. La clase cuenta con pocos métodos, pero son los que de momento me parecieron necesarios y suficientes para lo que normalmente se hace.

Descargar la clase phpDataClass

La clase la he venido probando desde hace algún tiempo en varios proyectos y se le han hecho algunas mejoras. Cualquier error que ustedes encuentren o si tienen alguna sugerencia la pueden enviar por medio del formulario de contacto.

Métodos de la clase phpDataClass para bases de datos MySQL

Los siguientes son los métodos de la clase phpDataClass que permiten interactuar con bases de datos MySQL:

Método Descripción Ejemplo
constructor(servidor, usuario, contraseña, base de datos) Es la llamada inicial que hacemos al instanciar la clase. Todos los parámetros son requeridos. Los parámetros que se envían son utilizados para conectar a la base de datos MySQL. 
$clsDataClass=new phpDataClass('localhost','root','micontrasena','basededatos');
executeQuery(consulta SQL) Mediante este método podemos insertar, actualizar y eliminar datos de una tabla. El mismo método se utiliza para esas tres acciones. 
if($clsDataClass->executeQuery("INSERT INTO empleados VALUES(NULL,'Cali','Rojas')")){
	echo 'Empleado agregado correctamente';
}else{
	echo 'Error al agregar el empleado';
}
lastID() Obtiene el ID único del último registro insertado en la base de datos. Es útil, por ejemplo, para cuando estamos utilizando campos con AUTOINCREMENT. 
if($clsDataClass->executeQuery("INSERT INTO empleados VALUES(NULL,'Cali','Rojas')")){
	echo 'Empleado agregado correctamente. El ID asignado es '.$clsDataClass->lastID();
}else{
	echo 'Error al agregar el empleado';
}
affectedRows() Retorna la cantidad de registros afectados por una consulta. Por ejemplo, al eliminar o actualizar datos de una tabla. 
if($clsDataClass->executeQuery("DELETE FROM empleados WHERE EmpleadoID>10")){
	echo 'Registros eliminados correctamente. Filas afectadas: '.$clsDataClass->affectedRows();
}else{
	echo 'Error al eliminar los registros';
}
lastError() Por medio de este método podemos obtener el último error de una consulta MySQL que hayamos ejecutado anteriormente. 
if($clsDataClass->executeQuery("DELETE FROM empleados WHERE EmpleadoID!!!!10")){
	echo 'Empleado eliminado correctamente. Filas afectadas: '.$clsDataClass->affectedRows();
}else{
	echo 'Error detectado: '.$clsDataClass->lastError();
}
rowCount(consulta SQL) Obtiene la cantidad de registros devueltos por la consulta que le pasamos. 
echo 'Empleados registrados: '.$clsDataClass->rowCount('SELECT * FROM empleados');
uniqueValue(consulta SQL) Sirve para obtener el primer registro o un único campo devuelto por la consulta que le pasemos. 
echo 'El usuario con ID 10 se llama: '.$clsDataClass->uniqueValue("SELECT Nombre + ' ' + Apellidos FROM empleados WHERE EmpleadoID=10");
getConnection() Esta clase maneja la conexión a la base de datos de forma automatizada, al realizar la primer consulta SQL se conecta a la base de datos utilizando los credenciales que se le pasaron al constructor al instanciarse. En algunos casos necesitamos usar una conexión abierta para algo, o para otro script PHP. No es necesario crear una nueva conexión, ya que con el método getConnection() podemos reutilizar la conexión que la clase estableció. 
if(mysql_query("SELECT * FROM tabla",$clsDataClass->getConnection())){
	//lo que sea
}else{
	//error
}
recordsList(consulta SQL) Este método sirve para consultar varios registros a la vez o para consultar una línea que contenga más de un campo de la base de datos, como por ejemplo, traer todos los registros almacenados en una tabla. El método devuelve un Array asociativo y a los elementos se accede por medio del nombre del campo. 
if($objDatos=$clsDataClass->recordsList('SELECT EmpleadoID,Nombre,Apellidos FROM empleados')){
	if(count($objDatos)>0){
		foreach($objDatos as $objEmpleado){
			echo 'ID del empleado: '.$objEmpleado['EmpleadoID'].'<br>';
			echo 'Nombre: '.$objEmpleado['Nombre'].'<br>';
			echo 'Apellidos: '.$objEmpleado['Apellidos'];
			echo '<hr>';
		}
	}else{
		echo 'No se encontraron empleados';
	}
}else{
	echo $clsDataClass->lastError();
}
closeConnection() Cierra la conexión a la base de datos establecida por la clase phpDataClass. El método raramente se utiliza, pero está disponible por si fuera necesario cerrar la conexión en algún momento para ahorrar recursos. Es importante mencionar que cuando se destruye la instancia de la clase, ésta cierra la conexión a la base de datos automáticamente ;) 
$clsDataClass->closeConnection();
sanitizeIn(cadena) Método para realizar una limpieza básica de las cadenas que se envían a la base de datos, para evitar errores y posibles inyecciones. 
$strNombreMarca=$clsDataClass->sanitizeIn("Pepito's brand");
if($clsClass->executeQuery("INSERT INTO marcas VALUES(NULL,'$strNombreMarca')")){
	echo 'Marca insertada correctamente';
}else{
	echo $clsClass->lastError();
}
sanitizeOut(cadena) Invierte lo que hizo el método sanitizeIn(). Es decir, nos devuelve la cadena original que el usuario insertó (sin las modificaciones que realizó sanitizeIn()).

Aquí también vemos un ejemplo de cómo utilizar el método recordsList().

 
if($objMarca=$clsDataClass->recordsList('SELECT Nombre,Estado FROM marcas WHERE MarcaID=24')){
	if(count($objMarca)>0){
		echo 'Nombre de la marca: '.$clsDataClass->sanitizeOut($objMarca['Nombre']).'<br>';
		echo 'Estado actual: '.$clsDataClass->sanitizeOut($objMarca['Estado']);
	}else{
		echo 'La marca no existe!';
	}
}else{
	echo $clsDataClass->lastError();
}
